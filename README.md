# CTF-CIT-Writeup

This repository contains my writeups for challenges from CTF@CIT, documenting the complete thought process, tools used, and step-by-step solutions.

The goal of this repo is not just to show answers, but to break down how each challenge is approached and solved. Each writeup focuses on practical techniques, debugging steps, and mistakes encountered during the process.

What you'll find here:
Detailed explanations of challenges
Scripts and payloads used during solving
Tools, commands, and workflows
Learnings and alternative approaches

Cryptography
Web Exploitation
